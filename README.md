# web3-fundamentals.github.io

Welcome to the web3 fundamentals project. Our mission is to create a public collection of the fundamental concepts and ideas around web3 to help people get started quickly in this new space.

---

## How to contribute

This project is public and intended to improve over time as anyone can contribute with their knowledge, suggestions and improvements. The website is generate with [VuePress](https://vuepress.vuejs.org/guide/getting-started.html#prerequisites) so look at the docs over there to get familiar. PRs welcome. Happy contributing :)
